# **App Name**: FiyatVizyon

## Core Features:

- Product Input and Management: Users can add new product entries with details such as product name, cost, store selling price, and online selling price, as well as view and delete existing products.
- Profit Calculation Table Generation: Allows users to specify a desired profit margin, and the system automatically calculates the corresponding selling price based on the product's cost price, displaying the results in a structured table.
- Dynamic Comparison Tables: Enables the generation and side-by-side display of multiple profit calculation tables, allowing users to easily compare different pricing strategies and profit scenarios simultaneously.
- Smart Profit Margin Suggester: An AI tool that provides recommendations for an optimal starting profit margin percentage for products, considering their cost and potentially general market dynamics.
- Product Data Persistence: Ensures that all entered product data (name, cost, selling prices) is securely stored and retrieved when the application is used, allowing users to manage their product catalog over time.

## Style Guidelines:

- Light color scheme to promote clarity and readability, aligning with the precision required for financial analysis.
- Primary color: Muted Professional Blue (#45627E), conveying trust, stability, and intelligence for a business application.
- Background color: Very Light Blue-Grey (#EEF1F3), offering a clean and unobtrusive canvas for data presentation.
- Accent color: Vibrant Aqua (#4CD8ED), used to highlight key actions, interactive elements, and positive indicators, providing a subtle contrast and visual interest.
- Main typeface: 'Inter' (grotesque-style sans-serif) for both headlines and body text, chosen for its modern, objective, and highly legible characteristics, ensuring clarity in data display.
- Utilize a set of simple, line-based icons that maintain a clean and professional appearance, consistent with a data-centric application.
- Implement a clear, modular layout that efficiently organizes product input forms, data tables, and calculation results, emphasizing accessibility for comparisons with side-by-side table views.
- Incorporate subtle animations for feedback during data input, table generation, and product management actions to enhance user interaction without distraction.
